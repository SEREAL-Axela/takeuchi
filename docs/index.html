DIGITALIS

NFCカードと公式台帳で、デジタルに所有感を。

カードをかざすだけで本物のコンテンツが開く。

解決する課題

デジタルコンテンツを高く、気持ちよく売買したい

デジタルは無限に複製でき、手触りがなく所有感が伝わらない

限定性や二次流通の設計が難しく、価格が上がりにくい

サービス概要

DIGITALISカード。デジタル原本をカードと一対一で結びつける仕組み

スマホにカードをかざすと、本物のコンテンツが開き、公式台帳で所有者を確認できます。ファンは「持てる」喜びを、IP側はプレミアと価格設計を実現できます。

課題の解決策

デジタルデータをNFCカード1枚と一対一で結びつけ、公式台帳で所有者を保証。カードをかざすだけで本物のコンテンツが開く仕組み。
